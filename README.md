# AWS EC2 Server Monitoring Project

## Overview
This is a static web project that demonstrates an AWS EC2 Server Monitoring system using CloudWatch, Alarms, SNS, and IAM concepts. It is designed as a portfolio/demo project.

## Pages
- **index.html** → Landing page (Project overview)
- **ec2.html** → EC2 instance setup and configuration steps
- **monitoring.html** → CloudWatch metrics and dashboard (CPU monitoring chart)
- **alarms.html** → CloudWatch alarms and SNS notifications
- **iam.html** → IAM users, groups, and permissions setup

## Features
- Responsive AWS-style UI (HTML + CSS)
- Fixed navigation bar across all pages
- CloudWatch monitoring chart integration (Chart.js)
- IAM, EC2, and monitoring workflow explanation
- Professional light theme design
- Side-by-side and grid image layouts

## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (basic DOM + Chart.js)
- Chart.js CDN
